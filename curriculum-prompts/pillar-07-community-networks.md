# Pillar 07 — Community Networks

## CO-STAR System Prompt

Copy this prompt into your preferred AI tool. It will generate a personalised Week 1 action plan for building community resilience — local connections, mutual aid networks, communication, and skill sharing.

---

```
CONTEXT:
I'm building community network resilience. Here's my current situation:

- Location: [YOUR_SUBURB], [YOUR_STATE] — HOW_LONG_LIVED_HERE
- Dwelling type: [HOUSE / APARTMENT / UNIT]
- Current community connection: [KNOW_NO_NEIGHBOURS / KNOW_A_FEW / ACTIVE_IN_COMMUNITY]
- Local knowledge: [NONE / KNOW_LOCAL_AMENITIES / KNOW_LOCAL_RISKS / CONNECTED]
- Skills I can offer: [LIST — E.G. ORGANISING / ADMIN / CHILDCARE / GARDENING / TECH / COOKING / TRADES]
- Skills I'd like to learn from others: [LIST]
- Communication options: [PHONE / WHATSAPP / HAM_RADIO / CB / SATELLITE_MESSENGER / NONE]
- Transport: [CAR / PUBLIC_TRANSPORT / BICYCLE / NONE]
- Household: [SINGLE / COUPLE / FAMILY / SHARED — ANY_ISOLATION_CONCERNS]
- Completed Pillar 01 (Mindset & Systems): [YES / NO]
- Personality note: [I'M_INTROVERTED / I'M_SOCIAL / I'M_NEW_TO_THE_AREA]

OBJECTIVE:
Generate my personal Week 1 action plan for "Community Networks." Start with low-risk, low-pressure actions. If I'm introverted or new to the area, give me scripts for what to say. The goal is one real connection this week — not a full neighbourhood network. Map local assets first (what's within walking distance), then identify one connection opportunity.

STYLE:
Practical, socially-aware. Assume social anxiety where stated. Provide actual scripts and conversation starters.

TONE:
Warm and encouraging — community building is vulnerable work. Like a friend who's also introverted but has figured out how to connect.

AUDIENCE:
An Australian who may be isolated, new to their area, or just doesn't know where to start with neighbours.

RESPONSE FORMAT:
- Week 1 Goal (one sentence — must be achievable if you're introverted)
- Local asset map — 3 things to find within 1km ([TIME_ESTIMATE])
- Connection strategy — one low-friction way to meet a neighbour
- Script — what to actually say (fill-in-the-blank)
- What I can offer — how to position my skills as useful to neighbours
- One community-anchored activity (community garden, street library, tool library, Buy Nothing group)
- One activity to do with my kids/partner that builds community
- Follow-up for Week 2 — what comes next if Week 1 works
```

---

## What to expect

The plan meets you where you are socially. Introverts get scripts and low-pressure approaches. Extroverts get ideas for organising. New arrivals get local mapping tasks. Established residents get mutual aid and skill-sharing suggestions.
