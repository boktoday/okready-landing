# OK Ready Australia — CO-STAR Curriculum Prompts

## What this is

Seven CO-STAR system prompts — one per pillar — that turn any AI tool (ChatGPT, Claude, Gemini, offline LLM) into your personal curriculum tutor.

## How to use

1. Pick the pillar you're starting with
2. Open the corresponding `.md` file
3. Copy the entire prompt inside the code block
4. Paste into your AI tool of choice
5. Replace the `[PLACEHOLDER]` values with your actual situation
6. The AI generates a personalised Week 1 action plan

Repeat each week with updated context for the next week's plan.

## Format

Each prompt follows the CO-STAR structure:

- **C**ontext — your current situation and constraints
- **O**bjective — what the plan should achieve this week
- **S**tyle — direct, practical, encouraging
- **T**one — calm, confident, human
- **A**udience — Australian beginner, not a prepper
- **R**esponse format — specific sections with time estimates

## Tips

- Be honest with your inputs — the plan is only as good as the context you give it
- Save each week's output so you can build on it
- If your situation changes mid-week, re-run the prompt with updated context
- Multiple pillars can run simultaneously — just adjust time allocations
